HealthConnect SL

 Community Health Management System for Sierra Leone

HealthConnect SL is a Digital Public Good (DPG) designed to improve community healthcare delivery in Sierra Leone by digitizing patient records, appointment scheduling, immunization tracking, maternal healthcare monitoring, and health reporting.

The project was developed as the final project for *PROG 102 – Principles of Software Engineering* at *Limkokwing University of Creative Technology, Sierra Leone Campus

---

 Project Overview

HealthConnect SL addresses the challenges of paper-based healthcare systems by providing a secure web and mobile platform that enables healthcare workers and patients to manage health information efficiently.

The system supports both smartphone users and feature phone users through an SMS/USSD interface, making healthcare services accessible even in remote communities.

# Features

- Patient Registration
- Secure Login with PIN
- Appointment Booking
- Digital Health Records
- Immunization Tracking
- Maternal Health Monitoring
- Community Health Worker Portal
- Clinic Queue Management
- SMS Appointment Reminders
- Offline Data Synchronization
- Reporting Dashboard
- Role-Based Access Control
- Audit Logging
- CSV & PDF Report Export
